Course Project

Create a website for Ye Olde Bookshop

For the Web Development course, which covered HTML and CSS and a little bit of JQuery, we had to create a full website for a fictional client wanting to create a presence for their bookshop. Each of us were also given a different author and a book of theirs to focus on for an author page.

This included repsonive design for all pages for mobile, tablet and dekstop sizes. Another requirement was a hamburger menu for mobile size and the use of only one CSS file.

Totaling eight pages in the end it included:

  • Homepage
  • New Arrivals
  • Author/Book Page
  • Valuations Page
  • Showcase Page including a video
  • Restorations
  • About Us
  • Enquiries
