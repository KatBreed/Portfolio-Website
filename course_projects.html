Create a single website for three different Luthiers

Having now completed my Diploma, the course finished out with having to produce a single streamlined website for three different Luthiers. This was put together using AWS with Ubuntu, Silverstripe CMS, Postgresql, Apache, WinSCP, and PHP as well as the core HTML, CSS, and Javascript. Create a website for Ye Olde Bookshop

Early on a course covered the core HTML and CSS and a little bit of JQuery. It was not made to go live but we were tasked with creating a full website for a fictional client wanting to create a presence for their book shop and restoration services. Each of us were also given a different author and a book of theirs to focus on for an author page. This included responsive design for all pages for mobile, tablet and desktop sizes. Another requirement was a hamburger menu for mobile size and the use of only one CSS file.
