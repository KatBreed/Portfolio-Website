Course Projects

During my time completing the Diploma of Web Development and Design, I completed two full websites. One for a book restoration shop wanting to gain a presence that was not made live. This was made using core HTML, CSS, and a little JQuery. The Diploma finished with having to create a website to be made live for three Luthiers, consolidating their separate websites into a single streamlined platform.

Create a single website for three different Luthiers

Having now completed my Diploma, the final project required producing a single streamlined website for three different Luthiers. This was built using AWS, Ubuntu, SilverStripe CMS, PostgreSQL, Apache, WinSCP, and PHP, along with core HTML, CSS, and JavaScript. The project also involved designing, specifying website requirements, billing, and post-development testing.

Create a website for Ye Olde Bookshop

One of my earlier course projects focused on core HTML, CSS, and a bit of jQuery. Though it was not intended to go live, we were tasked with designing a website for a fictional bookshop specializing in restoration services. Each student was assigned an author and one of their books to feature on a dedicated author page. The project emphasized responsive design across mobile, tablet, and desktop, using a single CSS file and incorporating a hamburger menu for mobile navigation.
